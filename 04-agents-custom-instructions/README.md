# 에이전트 커스텀 지침

에이전트 커스텀 지침은 대화형 AI 모델이 사용자로부터 더 나은 응답을 제공하고, 특정 요구 사항이나 요구를 충족할 수 있도록 훈련될 수 있는 방법입니다. 이러한 지침은 사용자가 원하는 특정 목표를 달성하는 데 도움이 될 수 있습니다.

## 지침 생성 방법

지침은 명확하고 간결하게 작성해야 합니다. 다음은 효과적인 지침을 작성하기 위해 고려할 몇 가지 요소입니다:

1. **특정성**: 모델이 해야 할 일과 하지 말아야 할 일을 명확하게 정의합니다.
   - 예: "나는 기술적 질문에 대한 중립적인 답변을 제공하길 원하지만, 편향된 의견은 피해야 합니다."

2. **상황 설명**: 모델이 응답할 때 고려해야 할 특정 상황이나 배경 정보를 제공합니다.
   - 예: "내가 데이터를 분석하는 방법에 대한 피드백을 요청할 때, 나의 이전 작업을 고려해주세요."

3. **언어 스타일**: 모델이 사용하는 언어의 톤이나 스타일을 지정하는 것도 중요합니다.
   - 예: "친근하고 격려하는 톤을 사용해주세요."

## 예시

다음은 몇 가지 예시 지침입니다:

1. "사용자가 제시한 문제에 대한 구체적인 해결책을 제시하고 추가 질문을 유도하세요."
2. "비슷한 문제를 해결한 경험을 공유하여 사용자가 느끼는 불안을 경감하세요."
